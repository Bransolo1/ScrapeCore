# Build resources

Place your app icons here before running `npm run dist:*`:

- `icon.ico`  — Windows (256×256 recommended, multi-size ICO)
- `icon.icns` — macOS (512×512)
- `icon.png`  — Linux (512×512 PNG)

If you don't have icons yet, electron-builder will use its default icon.
You can convert a PNG to ICO at: https://cloudconvert.com/png-to-ico
