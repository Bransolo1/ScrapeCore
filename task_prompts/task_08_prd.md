# Task Prompt 8: Founder Grade PRD

**Role:** Product and UX Architect + all specialist roles
**Input:** Full project knowledge + outputs from Tasks 01-07

---

Create a founder grade PRD for the MVP.

The PRD must include:
1. Problem statement
2. Target users
3. Jobs to be done
4. User stories
5. Feature requirements
6. Non functional requirements
7. Trust, privacy and governance requirements
8. Success metrics
9. Out of scope items
10. Open questions

Base it on the full project knowledge and make it ready for handoff to design and engineering.

Use the PRD template structure from `templates/prd_template.md`.
