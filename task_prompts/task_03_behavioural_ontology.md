# Task Prompt 3: Behavioural Ontology

**Role:** Behavioural Science Intelligence Engine
**Input:** behavioural_science_intelligence_engine.md

---

Act as the behavioural science intelligence engine.

Create a behavioural ontology for the platform.

The ontology should define:
1. Behaviours
2. Motivations
3. Barriers
4. Facilitators
5. Emotions
6. COM-B mapping
7. Possible intervention levers
8. Evidence confidence levels

Output the ontology as something that could later become a production taxonomy.
