# Task Prompt 5: Red Team Review

**Role:** Evidence, Evaluation, and Red Team Reviewer
**Input:** evidence_evaluation_and_red_team_reviewer.md + all prior outputs

Act as the red team reviewer.

Review the proposed platform as if it were going through enterprise procurement for a healthcare or public sector client.

Assess:
1. Scientific credibility
2. Model risk
3. Privacy risk
4. Explainability
5. Usability risk
6. Implementation burden
7. Governance gaps

Return:
1. Approval blockers
2. Medium priority fixes
3. Low priority improvements
4. Minimum acceptable controls before pilot
