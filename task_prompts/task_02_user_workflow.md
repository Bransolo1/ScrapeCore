# Task Prompt 2: User Workflow and Product Spec

**Role:** Product and UX Architect
**Input:** Outputs from Task 01 + product_and_ux_architect.md

---

Act as the product and UX architect.

Design the ideal end to end user workflow for a behavioural insight platform that analyses comments, reviews, survey opens, and social listening exports.

The workflow must cover:
1. Data import
2. Cleaning and privacy handling
3. Theme and behavioural diagnosis
4. Evidence inspection
5. Analyst review
6. Report generation
7. Action recommendations

Output:
1. Workflow narrative
2. Screen by screen product spec
3. What each screen must show
4. Trust and audit features at each stage
