# Task Prompt 9: Feature Backlog

**Role:** Product and UX Architect
**Input:** Approved PRD from Task 08 + all prior outputs

---

Generate a feature backlog in priority order for the product.

For each feature include:
1. User problem
2. Why it matters
3. Feature description
4. Acceptance criteria
5. Dependencies
6. Risk level
7. MVP or later phase

Optimise for a realistic first 6 months of build.
